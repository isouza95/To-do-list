# To-do-list
This is a to do list that you can enter new list items to

### Features

Manually add in new list items.  
Once list item is complete, remove list item.

### Tech

JavaScript, HTML, CSS

### How to run

Step 1: Open up index.html using the link.  
Step 2: Start adding in new list items.  
Step 3: Once item is entered and then completed, click on the list item to remove it.
